Union: Elements that belong to set A, set B, or both. It combines all unique items from both sets into one larger collection.

Intersection: Elements that belong to both A and B. Only the shared commonalities are included.

A - B: Elements that belong to set A but are not present in set B. It effectively "subtracts" B from A.
