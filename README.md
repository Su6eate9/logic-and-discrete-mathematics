# Lógica e Matemática Discreta

Este repositório reúne materiais, códigos e aplicações para o estudo de lógica e matemática discreta, com foco em conceitos fundamentais, operações de conjuntos, algoritmos e aplicações interativas.

## Estrutura do Repositório

- **conjuntos/**: Aplicação web para operações de conjuntos (união, interseção, diferença, complementar, diferença simétrica, produto cartesiano, inclusão). Inclui interface moderna e documentação.
- **assets/**: Recursos visuais e exemplos de telas para aplicações.

## Principais Temas

- Teoria dos conjuntos
- Operações matemáticas entre conjuntos
- Algoritmos e lógica
- Visualização e experimentação interativa

## Como usar

1. Navegue até a pasta desejada.
2. Siga as instruções específicas de cada subdiretório (exemplo: conjuntos/README.md).
3. Para aplicações web, abra o arquivo HTML no navegador ou execute um servidor local.

## Tecnologias Utilizadas

- HTML5, CSS, JavaScript (React via CDN)
- Tailwind CSS
- Babel Standalone

## Objetivo

Facilitar o aprendizado de lógica e matemática discreta por meio de exemplos práticos, interfaces intuitivas e documentação clara.
